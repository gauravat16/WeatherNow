# WeatherNow

A shell script to display weather as a notification.

## How to use it?

  * Clone this repository.
  * chmod +x ../WeatherNow/WeatherNow.sh
  * ./WeatherNow.sh
  
  Tested on MacOS, Ubuntu, elementary OS.
